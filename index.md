# Kinesthetic Learning Activities Archive

**This is a static archive of the original Kinesthetic Learning Activities wiki.**

A kinesthetic learning activity (KLA) is a physically engaging classroom exercise. These activities might, for example, teach recursion by simulating the Towers of Hanoi with students instead of disks, or highlight the difference between pipelined and non-pipelined execution using a human assembly line.

This website is a resource of KLAs for the Computer Science Education community, preserved for posterity and continued use.
