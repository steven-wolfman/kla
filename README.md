## Kinesthetic Learning Activities Archive

This repository contains a static Jekyll site preserving the educational content from the original Kinesthetic Learning Activities wiki, which was a valuable resource for computer science educators.

### What's Inside

- **53 Activities**: Hands-on kinesthetic learning activities for CS concepts 
- **3 Workshops**: Conference materials from SIGCSE and other venues
- **18 Resources**: Templates, research papers, and supporting materials

### Site Features

- Organized into Jekyll collections for easy browsing
- Responsive design optimized for mobile and desktop
- Clean markdown format with proper cross-references
- Preserved attribution for original contributors
- Static hosting compatible with GitHub Pages

### Development

Built using Jekyll with GitHub Pages compatibility:

```bash
bundle install
bundle exec jekyll serve
```
